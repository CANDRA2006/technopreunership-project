## Deskripsi

Repository ini adalah setup pertama dan dibuat sebagai bagian dari mata kuliah **Technopreneurship** dengan tujuan menggabungkan teknologi dan inovasi bisnis untuk menciptakan solusi yang bermanfaat bagi masyarakat.
